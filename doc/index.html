Documentação da API

GET /activate_sandbox

Ativa o sandbox.

Resposta: { "message": "Sandbox ativado." }

Exemplo de uso em Python:

import requests

response = requests.get('http://localhost:5000/activate_sandbox')
print(response.json())

GET /connect

Tenta conectar ao banco de dados.

Resposta: { "message": "Conexão estabelecida e encerrada." }

Exemplo de uso em Python:

import requests

response = requests.get('http://localhost:5000/connect')
print(response.json())

POST /execute_sql

Executa um comando SQL.

Corpo: { "command": "SEU_COMANDO_SQL_AQUI" }
Resposta: { "result": [...] }

Exemplo de uso em Python:

import requests

command = {"command": "SELECT * FROM sua_tabela"}
response = requests.post('http://localhost:5000/execute_sql', json=command)
print(response.json())

GET /tables

Retorna uma lista de todas as tabelas no banco de dados.

Resposta: [ "tabela1", "tabela2", ... ]

Exemplo de uso em Python:

import requests

response = requests.get('http://localhost:5000/tables')
print(response.json())

DELETE /drop_tables

Exclui todas as tabelas do banco de dados.

Resposta: { "message": "Todas as tabelas foram excluídas com sucesso." }

Exemplo de uso em Python:

import requests

response = requests.delete('http://localhost:5000/drop_tables')
print(response.json())
